# datasheets/

Fichas técnicas y datos de fabricante.

Sube aquí (arrastrar en GitHub → "Add file" → "Upload files"):

- Ficha del **aislador** (p. ej. Novibra RAEM2500): rigidez dinámica `K_din`, cantidad de aisladores.
- Datos del **equipo rotativo**: peso total y/o por pata, velocidad de operación (RPM), fuerzas de operación y de run-down (F_op, F_rd) en cada dirección.
- Planos o esquemas de disposición de apoyos, si los tienes.

Formatos: PDF, imágenes, xlsx — todos válidos.
